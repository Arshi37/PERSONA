# Picker Wheel - Random Decision Maker

A modern, feature-rich web application that recreates the functionality of pickerwheel.com. Make random decisions with style using a beautiful, animated spinning wheel!

## Features

### 🎯 **Core Functionality**
- **Interactive Spinning Wheel**: Smooth HTML5 Canvas-based wheel with realistic physics
- **Customizable Options**: Add, edit, and remove wheel segments with your own choices
- **Random Selection**: Fair and truly random selection algorithm
- **Beautiful Animations**: Smooth spinning with easing and bounce effects

### 🎨 **Visual Design**
- **Modern UI**: Clean, responsive design with gradient backgrounds
- **Colorful Segments**: Automatically assigned vibrant colors for each option
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Confetti Animation**: Celebration confetti when a result is selected

### 🔧 **Advanced Features**
- **Sound Effects**: Optional spinning and winning sound effects using Web Audio API
- **Customizable Settings**: Adjust spin duration (2-5 seconds)
- **Local Storage**: Automatically saves your options and preferences
- **Export Function**: Save the wheel as a PNG image
- **Keyboard Shortcuts**: 
  - `Enter` to add items
  - `Spacebar` to spin the wheel

### 🎵 **Audio & Effects**
- **Spinning Sound**: Dynamic audio feedback during wheel rotation
- **Victory Sound**: Celebratory chord progression when selection is made
- **Toggle Controls**: Enable/disable sound effects and confetti

### 📱 **Responsive Design**
- **Mobile Optimized**: Touch-friendly interface for all screen sizes
- **Adaptive Layout**: Automatically adjusts wheel size based on device
- **Cross-Platform**: Works on all modern browsers

## How to Use

1. **Add Options**: Type your choices in the input field and click "Add" or press Enter
2. **Customize**: Use the default options or create your own list
3. **Spin**: Click the "SPIN" button or press Spacebar to make a random selection
4. **Enjoy**: Watch the wheel spin and celebrate your random choice!

## Technical Details

- **Pure JavaScript**: No external dependencies (except Font Awesome for icons)
- **HTML5 Canvas**: High-performance wheel rendering and animations
- **CSS3**: Modern styling with flexbox, grid, and animations
- **Web Audio API**: Native browser sound generation
- **LocalStorage**: Persistent data storage

## Files

- `index.html` - Main application structure
- `styles.css` - Complete styling and responsive design
- `script.js` - Full application logic and wheel mechanics

## Browser Support

Works on all modern browsers that support:
- HTML5 Canvas
- ES6 JavaScript features
- CSS Grid and Flexbox
- Web Audio API (for sound effects)

## Getting Started

Simply open `index.html` in your web browser to start using the picker wheel!
